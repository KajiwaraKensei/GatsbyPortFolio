> src/data の中にあるファイルの変数をいじるだけでポートフォリオサイトが作れます

### ✨ [Demo](https://kaji-pf.netlify.app/)

自分の環境に持ってくる。

```bash
git clone https://github.com/KajiwaraKensei/GatsbyPortFolio.git
yarn install
npm start
```

[localhost:8000](https://localhost:8000)

## build

```bash
npm run build
```
